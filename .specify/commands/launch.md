# Launch Flux

Starts the complete Flux stack with one command.

## What it does:
- Starts FastAPI backend on port 9127
- Starts React frontend on port 9243
- Opens browser automatically

## Usage:
```
/launch
```

## Implementation:
```bash
cd "${SPECIFY_PROJECT_ROOT}"
python main.py
```
