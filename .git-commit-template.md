# Git Commit Template for Mosaic Systems

**Standard Authorship Footer for ALL commits:**

```
Authored by: Mani Saint-Victor, MD for Mosaic Systems
Generated with, in part, with Claude
```

**Template Usage:**
```bash
git commit -m "$(cat <<'EOF'
[commit message content]

Authored by: Mani Saint-Victor, MD for Mosaic Systems
Generated with, in part, with Claude
EOF
)"
```

**Key Points:**
- Always use "Mosaic Systems" (capital M, lowercase osaic, capital S, uppercase YSTEMS)
- Always include both lines in the footer
- Use HEREDOC format for multi-line commits
- Maintain consistent formatting
